# Whitepace — SaaS Landing Page


🚀 Project Overview:
A high-performance, fully responsive SaaS landing page built with a focus on modular architecture and clean code principles.
This project demonstrates advanced CSS layout techniques, component-based HTML structuring, and seamless third-party library integration.


🛠 Tech Stack:
- Bundler: Vite — utilized for optimized builds and fast development workflow.
- Templating: EJS (Embedded JavaScript) — implemented to achieve a modular HTML structure, separating sections into independent, maintainable components. 
- Styling: SCSS — structured using BEM methodology for scalable and predictable CSS.
- Interactions: Swiper.js — configured with custom navigation logic and dynamic active-state styling.
- Assets: Optimized SVGs and responsive image handling via <picture> and background properties. 

💎 Key Engineering Features:
- Component-Based HTML: By leveraging EJS, the project maintains a clean index.html. Each section (Hero, Sponsors, Testimonials, etc.) is a standalone module, significantly improving code readability and debugging. 
- Advanced Layout & Positioning: Strategic use of Flexbox and Grid to manage complex designs without "hacky" positioning. Focus on fluid responsiveness from 320px to 1440px+. 
- Performance & Security: Asset optimization and regular dependency audits. The project environment is maintained with 0 vulnerabilities for a secure development experience. 
- Custom Slider Integration: Tailored Swiper.js implementation, including synchronized background transitions and enhanced touch-target areas for mobile users.

💻 Getting Started
To run this project locally:

1. Clone the repository:
```git clone https://github.com/your-username/whitepace-project.git```

2. Install dependencies:
```npm install```


3. Launch the development server:
```npm run dev```
